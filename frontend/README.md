# Frontend

Interfaz inicial del MVP para `agente ia prueba`.

## Ejecutar localmente

Requiere Node.js y npm. Si no estan instalados globalmente, el proyecto puede usar el runtime portable ubicado en `.tools`.

```powershell
.\scripts\start-frontend.cmd
```

URL local:

```text
http://127.0.0.1:5173
```

Variable principal:

```text
VITE_API_BASE_URL=http://127.0.0.1:8000
```
