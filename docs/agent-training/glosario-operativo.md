# Glosario BPM, BPMN y Process Mining

## BPM
Disciplina de gestion que identifica, modela, analiza, mejora, ejecuta y monitorea procesos de negocio de extremo a extremo.

## BPMN
Notacion estandar para representar procesos de negocio mediante eventos, actividades, gateways, flujos, pools y lanes.

## As-is
Descripcion del proceso actual, con actividades, actores, sistemas, reglas, datos, excepciones y problemas existentes.

## To-be
Diseno del proceso futuro propuesto, validado con responsables y sustentado en evidencia.

## Process discovery
Fase de descubrimiento del proceso mediante entrevistas, talleres, documentos, observacion y datos.

## Process mining
Analisis de procesos a partir de event logs para descubrir modelos, comparar conformidad y medir performance.

## Event log
Registro de eventos con caso, actividad, timestamp y atributos usados para analizar ejecucion real del proceso.

## Gateway
Elemento BPMN usado para representar decisiones, paralelismo o convergencias en el flujo.

## Lane
Subdivision dentro de un pool BPMN que representa rol, area o participante responsable.

## Conformance checking
Comparacion entre comportamiento observado en datos y modelo esperado del proceso.

## Cycle time
Tiempo total de ejecucion desde inicio hasta fin de un caso o actividad.

## Bottleneck
Punto del flujo donde se acumula espera, capacidad insuficiente o demora relevante.
